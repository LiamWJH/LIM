# Stdlib.md - an intuitive guide to use the standard library

## print()
### usage
takes in unlimited amount of arguments that will be printed with a newline character.
### param types
All types
## scan()
### usage
takes in unlimited amounts of arguments but only the first will be prompted in the terminal and be returned.
### param types
All types
## length()
### usage
takes in one argument and returns the length,
### param types
String, Array
## type()
### usage
takes in one argument and returns the given argument's type.
### param types
All types
## biggest()
### usage
takes in unlimted amounts of arguments and returns the biggest one
### param types
Number
## smallest()
### usage
takes in unlimted amounts of arguments and returns the smallest one
### param types
Number
## append()
### usage
takes in two arguments each being an Array, and any item. Which then returns an Array with the item appended to the Array.
### param types
(1)Array (2)All types
